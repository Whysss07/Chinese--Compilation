# 中文 DSL（正式解释器版）

现在这个项目已经不是“字符串替换式”的小脚本了，而是一个真正的最小语言实现，包含：

- 词法分析器 `Lexer`
- 语法分析器 `Parser`
- AST 抽象语法树
- 解释执行器 `Interpreter`
- Python 转换器 `PythonTranspiler`

## 1. 当前支持的语法

### 变量

```txt
令 数字 = 10
名字 = "张三"
```

### 输出 / 输入

```txt
输出("你好")
令 名字 = 输入("请输入名字：")
```

### 分支

```txt
如果（分数 大于等于 90）
{
    输出("优秀")
}
否则如果（分数 大于等于 60）
{
    输出("及格")
}
否则
{
    输出("不及格")
}
```

### 循环

```txt
重复（3）
{
    输出("执行一次")
}

当（数字 < 5）
{
    数字 = 数字 + 1
}

对于（名字 在 列表("张三", "李四")）
{
    输出(名字)
}
```

### 函数

```txt
定义 求和(a, b)
{
    返回(a + b)
}
```

### 数组 / 列表 / 字典

```txt
令 分数 = 数组(90, 80, 70)
令 名单 = 列表("张三", "李四")
令 用户 = 字典(姓名="小明", 年龄=18)

令 数组 = [1, 2, 3]
令 信息 = {"姓名": "小明", "年龄": 18}
```

### 数组下标访问与赋值

```txt
令 数组 = [10, 20, 30]
输出(数组[0])
数组[1] = 999
追加(数组, 40)
输出(数组)
```

### 逻辑表达式

支持：

- `真 / 假 / 空`
- `且 / 并且`
- `或 / 或者`
- `非`
- `大于 / 小于 / 等于 / 不等于 / 大于等于 / 小于等于`

## 2. 运行

### 直接运行

```powershell
python .\zhdsl.py .\示例.zh
```

### 查看 AST

```powershell
python .\zhdsl.py .\示例.zh --show-ast
```

### 查看转换后的 Python

```powershell
python .\zhdsl.py .\示例.zh --show-python
```

---

## 3. 架构说明

执行流程：

```txt
DSL 源码
  -> Lexer 分词
  -> Parser 生成 AST
  -> Interpreter 解释执行
  -> （可选）Transpiler 转成 Python
```

也就是说，现在它已经具备“真正语言前端”的基本雏形了。

---

## 4. 当前限制

目前仍然是最小可用语言，不是完整工业级编程语言，因此还有这些限制：

- 暂不支持类和对象
- 暂不支持模块导入
- 暂不支持异常处理
- 自定义函数暂不支持命名参数调用
- 暂不支持对象属性赋值
- 更复杂的数组切片语法还没实现
