# 中文 DSL（最小可用版）

这是一个可直接用中文编写的最小 DSL，实现了：

- 顺序结构
- 分支结构
- 循环结构

底层实现方式是：**先把中文 DSL 翻译成 Python，再执行**。

---

## 1. 支持的语法

### 变量

```txt
令 数字 = 10
名字 = "张三"
```

### 输出

```txt
输出("你好")
输出("当前值 =", 数字)
```

### if / else if / else

```txt
如果（数字 > 10）
{
    输出("大于 10")
}
否则如果（数字 == 10）
{
    输出("等于 10")
}
否则
{
    输出("小于 10")
}
```

### while 循环

```txt
当（数字 < 5）
{
    输出(数字)
    数字 = 数字 + 1
}
```

### repeat 循环

```txt
重复（3）
{
    输出("执行一次")
}
```

---

## 4. 关键词对照

| 中文 DSL | 对应含义 |
|---|---|
| `如果(...)` | `if (...)` |
| `否则如果(...)` | `elif (...)` |
| `否则` | `else` |
| `当(...)` | `while (...)` |
| `重复(...)` | `for _ in range(...)` |
| `输出(...)` | `print(...)` |
| `令 变量 = 值` | 变量定义/赋值 |

## 5. 当前限制

这是一个**最小可用版本**，故意只保留核心能力：

- 主要支持顺序、分支、循环
- 表达式基本沿用 Python 写法，例如：`数字 < 10`、`a + b`
- 代码块必须使用 `{}` 包起来
- 暂未实现函数、类、模块、数组专用语法、异常处理等高级特性
